# Packages
## file-icons 
  让图标变的好看，即插即用
## Kite
  强大的python文档和IDE工具，需要额外安装Kite
## Hydrogen
  神器，谁用谁知道  
  配置：Languages Mappings：{ "python3": "magicpython" }
## minimap
  多一个代码略缩图，方便移动
## scipt
  直接执行代码，很方便
## platformio-ide-terminal
  终端也可以在Atom中打开，很方便
